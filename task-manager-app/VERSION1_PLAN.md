# Version 1 Build Plan

## Goal
Build a simple full-stack Task Manager App with core CRUD features.

## Build Order
1. Set up the frontend React app
2. Set up the backend Node.js and Express app
3. Connect backend to MongoDB
4. Create the Task model
5. Create API routes
6. Test API with sample data
7. Connect frontend to backend with Axios
8. Display tasks on the page
9. Add a new task
10. Mark a task as completed
11. Delete a task
12. Clean up the UI

## Version 1 Deliverable
A working task manager app where users can add, view, complete, and delete tasks.
