# Task Manager App

This project will be a full-stack task management application.

## Purpose
The app will help users create, update, organize, and track daily tasks.

## Planned Features for Version 1
- Add a new task
- View all tasks in one place
- Mark a task as completed
- Delete a task

## Planned Features for Version 2
- Edit a task
- Search tasks
- Filter tasks by completed or pending status

## Version 1 Frontend
- Single main page
- Task input form
- Task list display
- Complete task button
- Delete task button

## Version 1 Backend API
- GET /tasks
- POST /tasks
- PUT /tasks/:id
- DELETE /tasks/:id 

## Tech Stack
- React
- Node.js
- Express
- MongoDB

## Goal
To build a practical full-stack project that demonstrates CRUD operations, frontend and backend integration, and database usage.
