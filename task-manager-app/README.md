# Task Manager App

This project will be a full-stack task management application.

## Purpose
The app will help users create, update, organize, and track daily tasks.

## Planned Features
- Add a new task
- Edit a task
- Delete a task
- Mark a task as completed
- View all tasks in one place

## Tech Stack
- React
- Node.js
- Express
- MongoDB

## Goal
To build a practical full-stack project that demonstrates CRUD operations, frontend and backend integration, and database usage.
