# Backend

This folder will contain the Node.js and Express backend for the Task Manager App.

## Version 1 Responsibilities
- Get all tasks
- Create a new task
- Update task completion status
- Delete a task

## Version 1 API Endpoints
- GET /tasks
- POST /tasks
- PUT /tasks/:id
- DELETE /tasks/:id

## Main Tools
- Node.js
- Express
- MongoDB
- Mongoose
